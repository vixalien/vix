Vix Buttons Design

When you want to include links that capture more user attention or aren't in a paragraph or header, You use buttons which are also useful ways to provide access to scripts.

Block Button

A block button is one that describe the typical action a user may do. It is a main button on the page that the user may perform. Only one button of this type may be used in a context such as a form submit or Accept in a modal or Create Account in a navigation bar. It is the default button type.

This button use the primary color as background and the background color as text color. We assume that the primary color is dark violet. Remember contrast issues.

Outline Button

An outline button aka secondary button is another button that describe an action that subsidies a block button, However it is not very good to use it with a block button. This button has a primary-colored border and text and transparent background.

Text Button

A text button is a link button, It resembles a link except it is not inline and It emphasize a primary color rather than the usual link color.

Vix Links Design

Links are clickable text that directs a user where he want to go.

It is useful that links inherit font family, size, weight, etc from their parents.

Useful Guidelines for links

  • Keep the link's text short, Don't give long descriptions as links.
  • Use links for links, not scripts unless links are fallbacks when the link isn't available.
  • Use descriptive text such as Read More or Download

Don't

  • Use Link Text for links such as Click Here

Examples

  • Unvisited link
  • Unvisited hovered link
  • Unvisited active link
  • Visited link
  • Visited hovered link
  • Visited active link

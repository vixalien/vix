Headings

A heading acts as a title for the content that immediately follows it. We've declared default sizes for headings in keeping with their standard ranking but this base size can be overridden with more context-specific styling.

h1

When zombies arrive, quickly fax judge Pat.

h2

When zombies arrive, quickly fax judge Pat.

h3

When zombies arrive, quickly fax judge Pat.

h4

When zombies arrive, quickly fax judge Pat.

h5
When zombies arrive, quickly fax judge Pat.
h6
When zombies arrive, quickly fax judge Pat.
heading
When zombies arrive, quickly fax judge Pat.
  • h1 is for very important titles. Try not to use. When used, there must be only one for the whole page or for subsequent very important titles. H1s are not displayed on most screen-readers.
  • h2 was conned as the "usual" heading. It is mainly used as title of the page.
  • h3 as another heading level, used to show (not so) important headings, for example the website's name on a certain blog. Or use it for top navigation entities.
  • Use the h4 to divide your page into sections that one can read separately.
  • You may use h5 to provide additional content, for example examples and references must have a h5 heading.
  • h6 can be thought of as no heading a all, You can use as a caption for important images or subsections.
  • Use heading to caption images which are less important.

hr

This is hr, or horizontal rule our best friend who is multipurpose, whether to separate list items or apply a top or bottom border.

Table

Tables are ways to display data where you have many data with related fields where each field has many fields. Examples are displaying users with names, e-mails, and telephone numbers. We see that there are a list of users, each user, in turn having a list made of a name, e-mail and a telephone.

To display tables, only headings must be highlighted.

Blockquote

A blockquote or simply a quote aka excerpt is a length of words taken from somewhere. It should always go with a cite which shows from where it originated and maybe a possible link to the site.

Vix Layout is a simple way to organize different views within a page.

Container

A container is a box with free space on the left and right sides. Container's side's padding vary according to the device's width. A container is only supposed to have one element. On Smaller devices, the space becomes smaller significantly.

Flex

A flex is an array (list) of elements displayed on order, whether vertically or horizontally.

Flex is not only an array, it has much more control! You can; display the items as columns or rows, make it go in reverse order and make some items take more space than others.

For example if you have 6 elements and you set one item to flex, it will span all the available space and other elements will thrive with the available space. A flex can have tab spaces which divide the items. Following is an example of a use of flex and containers in a navbar.
